## TASK 1 - שיפורים אפשריים בהמשך

```text
You are working in the iris-demo-site repository.

Goal:
Implement one or more small, non-disruptive enhancements for the Iris demo app, only if the MVP is already stable.

Possible enhancements:
- show class probabilities
- add prepared sample inputs
- add a lightweight JSON API endpoint
- add simple explanatory text for the result
- add a reset workflow
- add basic visual polish

Rules:
- Prefer small enhancements with clear demo value.
- Keep each enhancement isolated and reversible.
- Do not introduce heavy architecture changes.
- Do not add databases, authentication, or background jobs.

Before coding:
- Inspect the current repo and choose only enhancements that fit the existing implementation cleanly.

At the end:
- List exactly which optional enhancements were implemented and which were intentionally skipped.
```
